# prometheus
By using this helm chart you can directly deploy the promethus and alertmanger which are from the operator mode of working 

you can add this repo by bellow commands
```
helm repo add prom https://mohanagandlla.github.io/prometheus/
helm repo update
```
Note:
Prometheus operator and CRDS should we installed
if you want to install the prometheus operator then follow this link 
https://github.com/prometheus-community/helm-charts/tree/main/charts/kube-prometheus-stack
